# **App Name**: MineServeHQ

## Core Features:

- Home Page: Landing page with server information and modern design.
- Navigation Bar: Navigation bar with links to 'Normativas' and 'Tienda' sections.
- Normativas Section: Normativas section with buttons for General, Policial, Ilicita, Medica, and Juridica, each leading to an informative page.
- User Authentication: User registration and secure login functionality.
- Tienda Section: Tienda section with item listings (image, description, terms of use, buy button).
- Payment Processing: Secure PayPal integration for item purchases.
- User Profiles: User profiles to track purchased items.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to create a modern and sophisticated atmosphere, reminiscent of a digital, futuristic world.
- Background color: Light gray (#F0F0F0) providing a subtle contrast that makes the content stand out, aligning with a clean design.
- Accent color: Electric Indigo (#7B1FA2) to draw attention to calls to action.
- Body and headline font: 'Inter' sans-serif, known for its clear, readable text.
- Modern and minimalistic icons for navigation and interactive elements.
- Clean and responsive layout that adapts to different screen sizes.
- Subtle animations to enhance user experience (e.g., button hover effects).
